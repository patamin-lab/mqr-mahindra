# .claude/prompts/

Parameterized prompt templates for requests that recur often enough to be
worth writing once. Fill in the bracketed placeholders and hand the result
to a Claude/agent session.

## Index

- `new-admin-resource.md` — add a new admin CRUD entity
- `new-business-module.md` — stand up a brand-new module under `modules/`
- `new-api-route.md` — add a single new API route to an existing area
- `vercel-serverless-diagnosis.md` — diagnose a production-only failure
